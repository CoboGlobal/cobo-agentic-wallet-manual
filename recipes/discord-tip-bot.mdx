## Business Context

Your Discord bot needs to reward contributors quickly, but a compromised prompt or bot bug can drain funds. You need a transfer flow that stays usable for micro-tips while hard-stopping risky amounts.

Core flow:

- Agent submits a pact requesting transfer rights with tip-sized limits
- Owner approves in the Web Console
- Bot submits one normal tip (allow)
- Bot submits oversized tip (deny with suggestion)
- Bot retries with compliant amount from the denial response and logs the outcome

<Accordion title="Policy Architecture">
```text
Owner (human)
  -> Delegation
       - Operator: bot principal (UUID)
       - Permissions: [write:transfer]
       - Policies:
           - max_per_tx: 5
           - daily_cumulative: 50
           - monthly_cumulative: 200
```
</Accordion>

## Pact Specification

The agent submits this PactSpec when requesting authorization. The owner reviews the tip cap, daily budget, and monthly ceiling before approving.

```json
{
  "permissions": ["write:transfer", "read:wallet"],
  "policies": [
    {
      "name": "deny-over-tip-limits",
      "type": "transfer",
      "rules": {
        "effect": "deny",
        "when": {},
        "deny_if": {
          "amount_usd_gt": "5",
          "usage_limits": {
            "rolling_24h": { "amount_usd_gt": "50" },
            "rolling_30d": { "amount_usd_gt": "200" }
          }
        }
      }
    }
  ],
  "duration_seconds": 2592000
}
```

## Full Flow

### Step 1: Owner sets the task

The owner gives the bot a natural-language instruction:

> "Run the Discord tip bot. Max $5 per tip, $50/day, $200/month. Valid 30 days."

### Step 2: Agent submits authorization request

The agent parses the instruction and submits a pact for owner approval.

import asyncio, os
from cobo_agentic_wallet import WalletAPIClient

async def main():
    async with WalletAPIClient(api_key=os.environ["AGENT_WALLET_API_KEY"]) as client:
        wallet_id = os.environ["AGENT_WALLET_WALLET_ID"]

        # Submit pact
        result = await client.submit_pact(
            wallet_id=wallet_id,
            intent="Run Discord tip bot: max $5/tip, $50 daily, $200 monthly, valid 30 days",
            spec={
                "permissions": ["write:transfer", "read:wallet"],
                "policies": [
                    {
                        "name": "deny-over-tip-limits",
                        "type": "transfer",
                        "rules": {
                            "effect": "deny",
                            "when": {},
                            "deny_if": {
                                "amount_usd_gt": "5",
                                "usage_limits": {
                                    "rolling_24h": {"amount_usd_gt": "50"},
                                    "rolling_30d":  {"amount_usd_gt": "200"},
                                },
                            },
                        },
                    },
                ],
                "duration_seconds": 2592000,
            },
        )
        pact_id = result["pact_id"]
        print(f"Pact {pact_id} submitted — awaiting owner approval in Web Console")

        # Poll for activation
        while True:
            pact = await client.get_pact(pact_id)
            if pact["status"] == "active":
                break
            await asyncio.sleep(10)

        print(f"Pact active. Scoped API key: {pact['api_key']}")

asyncio.run(main())

### Step 3: Owner approves in Web Console

The pact appears in the Web Console with the intent, execution plan, per-tip cap, daily limit, and monthly ceiling. The owner reviews and approves. The bot receives a pact-scoped API key and can begin sending tips.

### Step 4: Agent operates

With an active pact, the bot sends tips within the approved limits. When a tip exceeds the policy, the denial response includes the maximum allowed amount — the bot uses this to self-correct and retry without policy guesswork.

WALLET_UUID=<wallet_uuid>

# Allowed tip — within per-tip limit
caw tx transfer "$WALLET_UUID" --to "$DST" --token SETH_USDC --amount 3 --chain SETH

caw tx transfer "$WALLET_UUID" --to "$DST" --token SETH_USDC --amount 50 --chain SETH

# Self-corrected retry using the suggested compliant amount
caw tx transfer "$WALLET_UUID" --to "$DST" --token SETH_USDC --amount 5 --chain SETH

import asyncio, os
from cobo_agentic_wallet import WalletAPIClient

async def main():
    # Use the pact-scoped API key returned after approval
    async with WalletAPIClient(api_key=os.environ["PACT_API_KEY"]) as pact_client:
        wallet_id = os.environ["AGENT_WALLET_WALLET_ID"]

        # Allowed tip — within per-tip limit
        result = await pact_client.transfer_tokens(
            wallet_id=wallet_id,
            chain_id="SETH",
            dst_addr="0x1111111111111111111111111111111111111111",
            token_id="SETH_USDC",
            amount="3",
        )
        print(f"Tip submitted: {result['tx_id']}")

        # Oversized tip — policy denial includes suggested max amount
        try:
            await pact_client.transfer_tokens(
                wallet_id=wallet_id,
                chain_id="SETH",
                dst_addr="0x1111111111111111111111111111111111111111",
                token_id="SETH_USDC",
                amount="50",
            )
        except Exception as e:
            # e.suggestion contains the compliant amount — use it to self-correct
            print(f"Denied: {e}. Retrying with suggested amount.")
            result = await pact_client.transfer_tokens(
                wallet_id=wallet_id,
                chain_id="SETH",
                dst_addr="0x1111111111111111111111111111111111111111",
                token_id="SETH_USDC",
                amount="5",
            )
            print(f"Corrected tip submitted: {result['tx_id']}")

asyncio.run(main())

import { WalletAPIClient } from 'cobo-agentic-wallet';

// Use the pact-scoped API key returned after approval
const pactClient = new WalletAPIClient({ apiKey: process.env.PACT_API_KEY! });
const walletId = process.env.AGENT_WALLET_WALLET_ID!;
const dst = '0x1111111111111111111111111111111111111111';

// Allowed tip — within per-tip limit
const tipResult = await pactClient.transferTokens({
  wallet_id: walletId,
  chain_id: 'SETH',
  dst_addr: dst,
  token_id: 'SETH_USDC',
  amount: '3',
});
console.log(`Tip submitted: ${tipResult.tx_id}`);

// Oversized tip — denial response contains suggested compliant amount
try {
  await pactClient.transferTokens({
    wallet_id: walletId,
    chain_id: 'SETH',
    dst_addr: dst,
    token_id: 'SETH_USDC',
    amount: '50',
  });
} catch (e: any) {
  // e.suggestion contains the max allowed — use it to self-correct
  console.log(`Denied: ${e.message}. Retrying with suggested amount.`);
  const retryResult = await pactClient.transferTokens({
    wallet_id: walletId,
    chain_id: 'SETH',
    dst_addr: dst,
    token_id: 'SETH_USDC',
    amount: '5',
  });
  console.log(`Corrected tip submitted: ${retryResult.tx_id}`);
}
