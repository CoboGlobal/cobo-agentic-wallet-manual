title: "Manage Wallets and Addresses"
description: "Create wallets, add addresses, check balances, and manage wallet lifecycle."

Wallets are created and owned by a principal. The owner has unrestricted access; other principals access the wallet only via delegation.

## Create a wallet

<Tabs>
<Tab title="CLI">
```bash
caw wallet create --name "My Trading Wallet"
```
</Tab>
<Tab title="Python SDK">
```python
from cobo_agentic_wallet import WalletAPIClient

owner_client = WalletAPIClient(base_url=API_URL, api_key=OWNER_API_KEY)

wallet = await owner_client.create_wallet(
    name="Treasury",
)
wallet_uuid = wallet["uuid"]
print("Wallet UUID:", wallet_uuid)
```
</Tab>
<Tab title="TypeScript SDK">
```typescript
const wallet = await client.createWallet({
  name: 'Treasury',
});
const walletUuid = wallet.uuid;
console.log('Wallet UUID:', walletUuid);
```
</Tab>
</Tabs>

Wallet creation provisions the on-chain wallet via the Cobo Agentic Wallet service in the background.


## Add an address

Each chain requires its own address entry before transfers can be made to or from that chain.

<Tabs>
<Tab title="CLI">
```bash
caw address create <WALLET_UUID> --chain SETH
```
</Tab>
<Tab title="Python SDK">
```python
address = await owner_client.create_wallet_address(
    wallet_uuid=wallet_uuid,
    chain_identifier="SETH",
)
print("Address:", address["address"])
```
</Tab>
<Tab title="TypeScript SDK">
```typescript
const address = await client.createWalletAddress({
  wallet_uuid: walletUuid,
  chain_identifier: 'SETH',
});
console.log('Address:', address.address);
```
</Tab>
</Tabs>

## List wallets

<Tabs>
<Tab title="CLI">
```bash
# List all wallets
caw wallet list --format table

# Include archived wallets
caw wallet list --include-archived --format table
```
</Tab>
<Tab title="Python SDK">
```python
# Owner sees all their wallets
wallets = await owner_client.list_wallets(limit=50, offset=0)

# Agent sees only wallets it has active delegations for
agent_client = WalletAPIClient(base_url=API_URL, api_key=AGENT_API_KEY)
wallets = await agent_client.list_wallets()
```
</Tab>
<Tab title="TypeScript SDK">
```typescript
// Owner sees all their wallets
const wallets = await client.listWallets({ limit: 50, offset: 0 });

// Agent sees only wallets it has active delegations for
const agentClient = new WalletAPIClient({ baseUrl: '...', apiKey: AGENT_API_KEY });
const agentWallets = await agentClient.listWallets();
```
</Tab>
</Tabs>


## List wallet addresses

<Tabs>
<Tab title="CLI">
```bash
caw address list <WALLET_UUID>
```
</Tab>
<Tab title="Python SDK">
```python
addresses = await owner_client.list_wallet_addresses(wallet_uuid)
for addr in addresses.get("items", []):
    print(addr["address"], addr["chain_id"])
```
</Tab>
<Tab title="TypeScript SDK">
```typescript
const addresses = await client.listWalletAddresses({ wallet_uuid: walletUuid });
for (const addr of addresses.items ?? []) {
  console.log(addr.address, addr.chain_id);
}
```
</Tab>
</Tabs>


## Check balances

<Tabs>
<Tab title="CLI">
```bash

```
</Tab>

</Tabs>

</Tabs>


## Archive a wallet

Archiving prevents all operations on a wallet without deleting it.

<Tabs>
<Tab title="CLI">
```bash
caw wallet archive <WALLET_UUID>
```
</Tab>
<Tab title="Python SDK">
```python
await owner_client.archive_wallet(wallet_uuid)
```
</Tab>
<Tab title="TypeScript SDK">
```typescript
await client.archiveWallet({ wallet_uuid: walletUuid });
```
</Tab>
</Tabs>

An archived wallet cannot be transacted with. Delegations on an archived wallet are inaccessible.


## Wallet status

| Status | Description |
|---|---|
| `active` | Normal operating state |
| `archived` | Frozen permanently; no operations permitted |
