The Web Console is a browser-based interface where you can learn how Cobo Agentic Wallet works and get answers to your questions from an AI assistant. It is the right place to start if you are exploring the product or need help getting oriented.

## What you can do

| Task | How |
|---|---|
| Ask questions about the product | The AI assistant answers questions about how Cobo Agentic Wallet works, what features are available, and how to get started |
| Get guided to the right setup | The assistant walks you through what you need — whether to download the CLI, install the skill, or get the Human App |

## When to use the Web Console

The Web Console is the right starting point when you are new to Cobo Agentic Wallet and want to understand the product before setting up your wallet.

For ongoing wallet management — setting policies, approving transactions, reviewing activity — use the **Human App**. That is where wallet owners do most of their work.
