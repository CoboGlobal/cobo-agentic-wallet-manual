description: "Add the Cobo Agentic Wallet skill to your agent — no code required."
---

The Skill is a `SKILL.md` file that tells a compatible agent (such as OpenClaw) how to use Cobo Agentic Wallet. Once installed, the agent immediately understands wallet operations, policies, and approvals without any additional configuration.

## Step 1: Get the skill file

Download `SKILL.md` from the Cobo Agentic Wallet repository:

```bash
curl -O https://raw.githubusercontent.com/CoboGlobal/cobo-agent-wallets/main/SKILL.md
```

## Step 2: Add the skill to your agent

Drop `SKILL.md` into your agent's skill directory or attach it according to your agent platform's instructions. For OpenClaw:

```bash
cp SKILL.md ~/.openclaw/skills/cobo-agentic-wallet.md
```

## Step 3: Set up your wallet

Follow the [Set Up Your Agentic Wallet](/products/agentic-wallet/manual/quickstart/set-up-agentic-wallet) guide to connect your agent to the Cobo Agentic Wallet service and configure your policies.

## What the agent can do

Once the skill is installed, your agent can:

- Check wallet balances
- Transfer tokens
- Call smart contracts
- Query transaction history
- Respond correctly to policy denials and approval requests

<Info>No SDK or code changes are needed. The skill works with any agent that supports skill files.</Info>
