## Overview

The TypeScript SDK provides typed API classes for every domain in the Cobo Agentic Wallet REST API. It is built on `axios` and works in Node.js and edge runtimes.

## Installation

```bash
npm install @cobo/agentic-wallet
```

## Basic usage

```typescript
import { Configuration, WalletsApi, TransactionsApi } from '@cobo/agentic-wallet';

const config = new Configuration({
  apiKey: 'your-api-key',
  basePath: 'https://api.agenticwallet.cobo.com',
});

const walletsApi = new WalletsApi(config);
const response = await walletsApi.listWallets();
const wallets = response.data.result;
```

## Constructor

Each API class accepts a `Configuration` object:

```typescript
import { Configuration } from '@cobo/agentic-wallet';

const config = new Configuration({
  apiKey: 'your-api-key',
  basePath: 'https://api.agenticwallet.cobo.com',
});
```

<ParamField path="apiKey" type="string">
  `X-API-Key` header value for authentication.
</ParamField>

<ParamField path="basePath" type="string">
  API base URL. Defaults to the value from the OpenAPI spec if omitted.
</ParamField>

<ParamField path="baseOptions" type="object">
  Default options passed to every axios request.
</ParamField>

## Wallet methods

```typescript
import { WalletsApi } from '@cobo/agentic-wallet';

const walletsApi = new WalletsApi(config);

// List wallets accessible to the authenticated principal
const { data } = await walletsApi.listWallets(
  undefined, // X_API_Key override (optional)
  0,         // offset
  50,        // limit
  false,     // include_archived
);

// Get a specific wallet
const wallet = await walletsApi.getWallet(walletUuid);

// Create a wallet
const newWallet = await walletsApi.createWallet({
  wallet_type: 'MPC-UCW',
  name: 'My Agent Wallet',
  main_node_id: 'tss-node-id',
});

// Update wallet name or metadata
await walletsApi.updateWallet(walletUuid, { name: 'Updated Name' });

// Archive a wallet
await walletsApi.archiveWallet(walletUuid);

// Create a new on-chain address
const address = await walletsApi.createWalletAddress(walletUuid, {
  chain_type: 'BASE_ETH',
});

// List addresses for a wallet
const addresses = await walletsApi.listWalletAddresses(walletUuid);

// Get the TSS node online status
const nodeStatus = await walletsApi.getWalletNodeStatus(walletUuid);
```

## Transaction methods

```typescript
import { TransactionsApi } from '@cobo/agentic-wallet';

const txApi = new TransactionsApi(config);

// Transfer tokens
const result = await txApi.transferTokens(walletUuid, {
  wallet_id: walletUuid,
  chain_id: 'SETH',
  dst_addr: '0xRecipient...',
  token_id: 'SETH_USDC',
  amount: '50',
  request_id: 'invoice-001',
});

// Estimate transfer fee
const fee = await txApi.estimateTransferFee(walletUuid, {
  dst_addr: '0xRecipient...',
  amount: '50',
  token_id: 'SETH_USDC',
});

// Smart contract call (EVM)
const callResult = await txApi.contractCall(walletUuid, {
  chain_id: 'BASE_ETH',
  contract_addr: '0xContract...',
  calldata: '0x...',
  value: '0',
  request_id: 'call-001',
});

// Estimate contract call fee
const callFee = await txApi.estimateContractCallFee(walletUuid, {
  chain_id: 'BASE_ETH',
  contract_addr: '0xContract...',
  calldata: '0x...',
});

// Sign a message or EIP-712 typed data
const sig = await txApi.messageSign(walletUuid, {
  chain_id: 'BASE_ETH',
  eip712_typed_data: { ... },
  sync: true,
});

// Process an x402 payment request
const payment = await txApi.payment(walletUuid, {
  protocol: 'x402',
  x402_payment_required: '<PAYMENT-REQUIRED header value>',
  request_id: 'payment-001',
});

// Drop (cancel) a pending transaction
await txApi.dropTransaction(walletUuid, {
  transaction_uuid: txUuid,
  request_id: 'drop-001',
});

// Speed up a pending transaction with a higher fee
await txApi.speedupTransaction(walletUuid, {
  transaction_uuid: txUuid,
  fee: { ... },
});

// List recently used destination addresses
const recent = await txApi.listRecentAddresses(walletUuid);
```

## Pact methods

```typescript
import { PactsApi } from '@cobo/agentic-wallet';

const pactsApi = new PactsApi(config);

// Submit a pact for owner approval
const result = await pactsApi.submitPact({
  wallet_id: 'a1b2c3d4-...',
  intent: 'DCA into ETH for 3 months',
  spec: {
    permissions: ['write:contract_call'],
    completion_conditions: [
      { type: 'time_elapsed', threshold: '7776000' },
    ],
  },
});
const pactId = result.data.result.pact_id;

// Get pact detail — triggers lazy activation if the approval was resolved
const pact = await pactsApi.getPact(pactId);
const apiKey = pact.data.result.api_key; // present when status === 'active'

// List pacts
const pacts = await pactsApi.listPacts(
  'active',  // status filter
  walletId,  // wallet_id filter
  0,         // offset
  50,        // limit
);

// Get pact lifecycle events
const events = await pactsApi.listPactEvents(pactId);

// Revoke an active pact
const revoked = await pactsApi.revokePact(pactId);
```

## Delegation methods

```typescript
import { DelegationsApi } from '@cobo/agentic-wallet';

const delegationsApi = new DelegationsApi(config);

// Create a delegation (grant operator access to a wallet)
const delegation = await delegationsApi.createDelegation({
  operator_id: 'agent-principal-id',
  wallet_id: walletUuid,
  permissions: ['write:transfer', 'read:wallet'],
  expires_at: '2026-12-31T00:00:00Z',
});

// Get a specific delegation
const d = await delegationsApi.getDelegation(delegationId);

// List delegations created by the authenticated principal
const delegations = await delegationsApi.listDelegations(
  undefined, // operator_id filter
  walletUuid,
  'active',  // status filter
  0,         // offset
  50,        // limit
);

// List delegations received (where you are the operator)
const received = await delegationsApi.listReceivedDelegations();

// Update a delegation
await delegationsApi.updateDelegation(delegationId, { ... });

// Revoke a delegation
await delegationsApi.revokeDelegation(delegationId);

// Freeze / unfreeze delegations
await delegationsApi.freezeDelegations({ scope: 'delegation', delegation_id: delegationId });
await delegationsApi.unfreezeDelegations({ scope: 'delegation', delegation_id: delegationId });
```

## Policy methods

```typescript
import { PoliciesApi } from '@cobo/agentic-wallet';

const policiesApi = new PoliciesApi(config);

// Create a policy
const policy = await policiesApi.createPolicy({
  scope: 'wallet',
  wallet_id: walletUuid,
  name: 'Transfer limit',
  policy_type: 'transfer',
  rules: { max_amount_per_tx: '100', token_id: 'SETH_USDC' },
  priority: 10,
  is_active: true,
});

// Get a specific policy
const p = await policiesApi.getPolicy(policyId);

// List policies
const policies = await policiesApi.listPolicies(
  'wallet',   // scope
  walletUuid, // wallet_id
  undefined,  // delegation_id
  'transfer', // policy_type
);

// Update a policy
await policiesApi.updatePolicy(policyId, { ... });

// Deactivate a policy
await policiesApi.deactivatePolicy(policyId);

// Dry-run policy evaluation
const evalResult = await policiesApi.dryRunPolicy({
  wallet_id: walletUuid,
  operation_type: 'transfer',
  amount: '500',
  chain_id: 'SETH',
  token_id: 'SETH_USDC',
  dst_addr: '0xRecipient...',
});
```

## Identity methods

```typescript
import { IdentityApi } from '@cobo/agentic-wallet';

const identityApi = new IdentityApi(config);

// Provision an agent using a pairing code
const provisioned = await identityApi.provisionAgent({ token: 'pairing-code' });

// Agent-first provisioning — no token
const agentFirst = await identityApi.provisionAgent({});
// agentFirst.data.result.activation_secret — share with the human

// Complete agent-first provisioning
await identityApi.activateAgent({
  agent_id: 'agent-uuid',
  owner_id: 'owner-principal-id',
  activation_secret: 'secret-from-provision',
});

// Get agent provisioning status
const agentStatus = await identityApi.getAgentStatus(agentId);

// List agents owned by the authenticated principal
const agents = await identityApi.listMyAgents();

// Create an API key for the authenticated principal
const apiKey = await identityApi.createApiKey({
  name: 'My key',
  scopes: ['*'],
});

// List API keys
const keys = await identityApi.listApiKeys();

// Revoke an API key
await identityApi.revokeApiKey(keyId);
```

## Audit methods

```typescript
import { AuditApi } from '@cobo/agentic-wallet';

const auditApi = new AuditApi(config);

// List audit logs with cursor-based pagination
const logs = await auditApi.listAuditLogs(
  undefined,   // X_API_Key override
  walletId,    // wallet_id filter
  undefined,   // principal_id filter
  undefined,   // action filter
  'denied',    // result filter
  undefined,   // start_time
  undefined,   // end_time
  undefined,   // cursor
  50,          // limit
);

// Pass next_cursor to fetch the next page
const nextPage = await auditApi.listAuditLogs(
  undefined, undefined, undefined, undefined, undefined,
  undefined, undefined,
  logs.data.result.next_cursor,
  50,
);
```
