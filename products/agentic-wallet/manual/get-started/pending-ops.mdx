## Transaction statuses

| Status | What it means |
|---|---|
| Completed | Transaction executed on-chain |
| Awaiting approval | Exceeded your approval threshold — waiting for you to decide |
| Blocked | Violated a rule you set and was stopped automatically |
| Pending on-chain | Submitted to the network but not yet confirmed |

## Approve or reject an over-limit transaction

If you set an approval threshold — for example, "require approval for transfers over $50" — any transaction your agent attempts above that amount pauses automatically and waits for your decision. Your agent cannot proceed until you act.

You'll receive a push notification in the Human App when a transaction is waiting. You can also check the **Pending Transactions** section at any time, or ask the AI assistant:

> "Do I have any transactions waiting for my approval?"

Each paused transaction shows the amount and token, the destination address, why it was paused, and when it was submitted.

- **Approve** — the transaction executes on-chain
- **Reject** — the transaction is cancelled; your agent receives a clear explanation and can resubmit within your limits if appropriate

<Warning>If you don't act within one hour, the transaction expires automatically. Your agent is notified and can resubmit if needed.</Warning>

## Speed up or drop a stuck transaction

A transaction shows **Pending on-chain** when it has been submitted to the blockchain but has not confirmed yet. This usually happens when network traffic is high and the gas fee attached to the transaction is too low for miners to prioritize it.

You have two options:

**Speed it up** — resubmit the transaction with a higher fee. The network treats this as a replacement and miners will pick it up faster. Ask the AI assistant:

> "Speed up the pending transaction on wallet X."

**Drop it** — cancel the transaction before it confirms. This pushes the original out of the queue. Use this if you no longer want the operation to execute:

> "Cancel the pending transaction on wallet X."

Both actions are available through the Human App AI assistant. If your agent submitted the transaction, you can also ask it directly.
