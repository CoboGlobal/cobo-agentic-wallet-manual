title: "Manage Wallet"

Wallets are created and owned by a Principal. The owner has unrestricted access; other principals access the wallet only via delegation.

## Create a wallet

```python
owner_client = WalletAPIClient(base_url=API_URL, api_key=OWNER_API_KEY)

wallet = await owner_client.create_wallet(
    name="Treasury",
    wallet_type="Custodial-Web3",  # or "MPC-Web3"
)
wallet_uuid = wallet["uuid"]
print("Wallet UUID:", wallet_uuid)
```

**Wallet types:**

| Type | Description |
|---|---|
| `Custodial-Web3` | Cobo-custodied wallet; Cobo holds the private key |
| `MPC-Web3` | MPC wallet; private key is threshold-shared |

Wallet creation provisions the on-chain wallet via the Cobo Agentic Wallet service in the background.


## Add an address

Each chain requires its own address entry before transfers can be made to/from that chain.

```python
address = await owner_client.create_address(
    wallet_uuid=wallet_uuid,
    chain_id="ETH",
)
print("Address:", address["address"])
```


## Get a wallet

```python
wallet = await owner_client.get_wallet(wallet_uuid)
print(wallet["name"], wallet["status"])
```


## List wallets

```python
# Owner sees all their wallets
wallets = await owner_client.list_wallets(limit=50, offset=0)

# Agent sees only wallets it has active delegations for
agent_client = WalletAPIClient(base_url=API_URL, api_key=AGENT_API_KEY)
wallets = await agent_client.list_wallets()
```


## List wallet addresses

```python
addresses = await owner_client.list_wallet_addresses(wallet_uuid, chain_id="ETH")
for addr in addresses.get("items", []):
    print(addr["address"], addr["chain_id"])
```


## Update wallet metadata

```python
await owner_client.update_wallet(wallet_uuid, name="Main Treasury", metadata={"env": "prod"})
```


## Archive a wallet

Archiving prevents all operations on a wallet without deleting it.

```python
await owner_client.archive_wallet(wallet_uuid)
```

An archived wallet cannot be transacted with. Delegations on an archived wallet are inaccessible.


## Wallet status

| Status | Description |
|---|---|
| `active` | Normal operating state |
| `archived` | Frozen permanently; no operations permitted |


## Get wallet spend status

Returns current spend counters for all active delegations on a wallet:

```python
status = await owner_client.get_wallet_status(wallet_uuid)
print(status)
# {
#   "wallet_id": "...",
#   "delegations": [
#     {
#       "delegation_id": "...",
#       "operator_id": "...",
#       "daily_spent": "245.00",
#       "daily_limit": "500.00",
#       "frozen": false
#     }
#   ]
# }
```
