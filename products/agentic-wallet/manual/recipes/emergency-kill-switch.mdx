## Business Context

When suspicious behavior appears, owners need a one-command control to stop all delegated operations.

<Accordion title="Control Architecture">
```
Owner (human)
  -> Delegation A (agent A)  ← backed by active pact
  -> Delegation B (agent B)  ← backed by active pact

Emergency action:
  caw delegation freeze --scope owner

Effect:
  - All active delegations become frozen
  - Agent transfer attempts are denied until owner intervention
```
</Accordion>

## Pact Specification

Before demonstrating the kill switch, the agent has an active pact. The owner approved this pact earlier, giving the agent scoped transfer access that the freeze command will block.

```json
{
  "permissions": ["write:transfer", "read:wallet"],
  "policies": [
    {
      "name": "deny-over-daily-limit",
      "type": "transfer",
      "rules": {
        "effect": "allow",
        "when": {},
        "deny_if": {
          "amount_usd_gt": "5"
        }
      }
    }
  ],
  "duration_seconds": 604800
}
```

## Full Flow

### Step 1: Agent establishes active pact

Before the emergency scenario, the agent requests and receives authorization via the pact flow.

```bash
# Write the PactSpec to a file
cat > agent-pact.json << 'EOF'
{
  "permissions": ["write:transfer", "read:wallet"],
  "policies": [
    {

      "type": "transfer",

        "effect": "allow",
        "when": {},
        "deny_if": { "amount_usd_gt": "5" }
      }
    }
  ],
  "duration_seconds": 604800
}
EOF

# Agent submits pact and waits for owner approval
caw pact submit \
  --intent "Operate agent with small transfer allowance: max \$5/tx, valid 7 days" \
  --execution-plan "See the pact spec JSON above." \
  --policies "$(python3 -c \"import json,sys; d=json.load(open('agent-pact.json')); print(json.dumps(d['policies']))\")" \
  --completion-conditions "$(python3 -c \"import json,sys; d=json.load(open('agent-pact.json')); print(json.dumps(d.get('completion_conditions', [])))\")"

# Confirm active
caw pact status <PACT_ID>
```

<Tab title="Python SDK">

```python
import asyncio, os
from cobo_agentic_wallet import WalletAPIClient

async def main():
    async with WalletAPIClient(api_key=os.environ["AGENT_WALLET_API_KEY"]) as client:
        wallet_id = os.environ["AGENT_WALLET_WALLET_ID"]

        # Agent submits pact
        result = await client.submit_pact(
            wallet_id=wallet_id,
            intent="Operate agent with small transfer allowance: max $5/tx, valid 7 days",
            spec={
                "permissions": ["write:transfer", "read:wallet"],
                "policies": [
                    {
                        "name": "deny-over-daily-limit",
                        "type": "transfer",
                        "rules": {
                            "effect": "allow",
                            "when": {},
                            "deny_if": {"amount_usd_gt": "5"},
                        },
                    },
                ],
                "duration_seconds": 604800,
            },
        )
        pact_id = result["pact_id"]
        print(f"Pact {pact_id} submitted — awaiting owner approval in Human App")

        # Poll for activation
        while True:
            pact = await client.get_pact(pact_id)
            if pact["status"] == "active":
                break
            await asyncio.sleep(10)

        print(f"Pact active. Agent is now operating.")

asyncio.run(main())
```

### Step 4: Owner detects suspicious activity and freezes

The owner notices unusual patterns and triggers the emergency kill switch.

<Note>
  Freeze and unfreeze operations are only available via the Python SDK. There are no `caw delegation freeze` or `caw delegation unfreeze` CLI commands.
</Note>

**Python SDK:**

```python
import asyncio, os
from cobo_agentic_wallet import WalletAPIClient

async def main():
    async with WalletAPIClient(api_key=os.environ["AGENT_WALLET_API_KEY"]) as client:
        # Freeze all delegations for this owner — immediate effect
        await client.freeze_delegations(scope="owner")
        print("All delegations frozen. Agent transfers are now denied.")

asyncio.run(main())
```

All active delegations, including those backed by approved pacts, are frozen immediately. Any subsequent agent transfer attempt is denied.

### Step 5: Owner unfreezes when safe

After investigating and resolving the incident, the owner restores agent access.

**Python SDK:**

```python
import asyncio, os
from cobo_agentic_wallet import WalletAPIClient

async def main():
    async with WalletAPIClient(api_key=os.environ["AGENT_WALLET_API_KEY"]) as client:
        # Unfreeze and restore agent access
        await client.unfreeze_delegations(scope="owner")
        print("Delegations unfrozen. Agent can operate again.")

asyncio.run(main())
```
