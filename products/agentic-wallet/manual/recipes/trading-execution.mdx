## Business Context

Trading automation must stay inside risk rails: capped order size, capped short-window exposure, and strict destination routing.

<Accordion title="Trading Risk Architecture">
```text
Owner (human)
  -> Execution Delegation (agent)
       - write:transfer
       - transfer risk caps:
           - max_per_tx_value
           - hourly_value_limit
           - daily_value_limit
       - address allowlist:
           - approved DEX routers only
```
</Accordion>

## Pact Specification

The agent submits this PactSpec when requesting authorization. The owner reviews permissions, spending caps, and the router allowlist before approving.

```json
{
  "permissions": ["write:transfer", "read:wallet"],
  "policies": [
    {
      "name": "allow-dex-routers",
      "type": "transfer",
      "rules": {
        "effect": "allow",
        "when": {
          "dst_in": [
            "0xRouter1...",
            "0xRouter2..."
          ]
        },
        "review_if": {
          "amount_usd_gt": "80"
        }
      }
    },
    {
      "name": "deny-over-limits",
      "type": "transfer",
      "rules": {
        "effect": "deny",
        "when": {},
        "deny_if": {
          "amount_usd_gt": "100",
          "usage_limits": {
            "rolling_1h":  { "amount_usd_gt": "500" },
            "rolling_24h": { "amount_usd_gt": "2000" }
          }
        }
      }
    }
  ],
  "duration_seconds": 2592000
}
```

## Full Flow

### Step 1: Owner sets the task

The owner gives the agent a natural-language instruction:

> "Run my trading execution bot. Max $100 per order, $500 hourly, $2,000 daily. Only approved DEX routers. Valid 30 days."

### Step 2: Agent submits pact

The agent parses the instruction and submits a pact for owner approval.

```python
import asyncio, os
from cobo_agentic_wallet import WalletAPIClient

async def main():
    async with WalletAPIClient(api_key=os.environ["AGENT_WALLET_API_KEY"]) as client:
        wallet_id = os.environ["AGENT_WALLET_WALLET_ID"]

        # Submit pact
        result = await client.submit_pact(
            wallet_id=wallet_id,
            intent=(
                "Run trading execution bot: max $100/order, $500/hour, "
                "$2,000/day, approved DEX routers only, valid 30 days"
            ),
            spec={
                "permissions": ["write:transfer", "read:wallet"],
                "policies": [
                    {
                        "name": "allow-dex-routers",
                        "type": "transfer",
                        "rules": {
                            "effect": "allow",
                            "when": {"dst_in": ["0xRouter1...", "0xRouter2..."]},
                            "review_if": {"amount_usd_gt": "80"},
                        },
                    },
                    {
                        "name": "deny-over-limits",
                        "type": "transfer",
                        "rules": {
                            "effect": "deny",
                            "when": {},
                            "deny_if": {
                                "amount_usd_gt": "100",
                                "usage_limits": {
                                    "rolling_1h":  {"amount_usd_gt": "500"},
                                    "rolling_24h": {"amount_usd_gt": "2000"},
                                },
                            },
                        },
                    },
                ],
                "duration_seconds": 2592000,
            },
        )
        pact_id = result["pact_id"]
        print(f"Pact {pact_id} submitted — awaiting owner approval in Human App")

        # Poll for activation
        while True:
            pact = await client.get_pact(pact_id)
            if pact["status"] == "active":
                break
            await asyncio.sleep(10)

        print(f"Pact active. Scoped API key: {pact['api_key']}")

asyncio.run(main())
```

### Step 3: Owner approves  in Human App

The pact appears in the Human App with the intent, execution plan, permissions, spending caps, and router allowlist. The owner reviews and approves. The agent receives a pact-scoped API key and can begin operating.

### Step 4: Agent operates

With an active pact, the agent executes trades within the approved limits.

<Tabs>

<Tab title="CLI">

```bash
WALLET_UUID=<wallet_uuid>
ROUTER=<approved_dex_router>

# Allowed order — within per-tx and hourly limits
caw tx transfer "$WALLET_UUID" --to "$ROUTER" --token SETH --amount 80 --chain SETH

# Oversized order — exceeds per-tx cap, triggers policy denial
caw tx transfer "$WALLET_UUID" --to "$ROUTER" --token SETH --amount 500 --chain SETH

# Retry with compliant amount
caw tx transfer "$WALLET_UUID" --to "$ROUTER" --token SETH --amount 100 --chain SETH
```
