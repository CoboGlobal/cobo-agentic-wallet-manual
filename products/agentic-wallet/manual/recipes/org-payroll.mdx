## Business Context

Payroll automation must handle routine payouts quickly while escalating larger transfers to owner approval and enforcing hard monthly ceilings.

<Accordion title="Tiered Payroll Control Architecture">
```text
Owner (human)
  -> Junior Ops Delegation
       - max_per_tx_value: 25
       - monthly_value_limit: 2000
       - recipient allowlist

  -> Senior Ops Delegation
       - max_per_tx_value: 500
       - approval_threshold_value: 300
       - monthly_value_limit: 500
       - recipient allowlist
```
</Accordion>

## Pact Specification

The agent submits this PactSpec when requesting payroll authorization. The single pact covers the full tiered policy: routine payouts execute automatically, while larger payouts are paused for owner review. The owner reviews recipient allowlist, per-payout cap, approval threshold, and monthly budget before approving.

```json
{
  "permissions": ["write:transfer", "read:wallet"],
  "policies": [
    {
      "name": "allow-payroll-recipients",
      "type": "transfer",
      "rules": {
        "effect": "allow",
        "when": {
          "dst_in": [
            "0xPayrollRecipient1...",
            "0xPayrollRecipient2...",
            "0xPayrollRecipient3..."
          ]
        },
        "review_if": {
          "amount_usd_gt": "300"
        }
      }
    },
    {
      "name": "deny-over-limits",
      "type": "transfer",
      "rules": {
        "effect": "deny",
        "when": {},
        "deny_if": {
          "amount_usd_gt": "500",
          "usage_limits": {
            "rolling_30d": { "amount_usd_gt": "5000" }
          }
        }
      }
    }
  ],
  "duration_seconds": 7776000
}
```

## Full Flow

### Step 1: Owner sets the task

The owner gives the agent a natural-language instruction:

> "Run payroll automation. Max $500 per payout, require my approval above $300, only approved payroll addresses, $5,000 monthly budget. Valid 90 days."

### Step 2: Agent submits pact

The agent parses the instruction and submits a pact for owner approval.

import asyncio, os
from cobo_agentic_wallet import WalletAPIClient

async def main():
    async with WalletAPIClient(api_key=os.environ["AGENT_WALLET_API_KEY"]) as client:
        wallet_id = os.environ["AGENT_WALLET_WALLET_ID"]

        # Submit pact
        result = await client.submit_pact(
            wallet_id=wallet_id,
            intent=(
                "Run payroll automation: max $500/payout, owner approval above $300, "
                "approved recipients only, $5,000 monthly, valid 90 days"
            ),
            spec={
                "permissions": ["write:transfer", "read:wallet"],
                "policies": [
                    {
                        "name": "allow-payroll-recipients",
                        "type": "transfer",
                        "rules": {
                            "effect": "allow",
                            "when": {
                                "dst_in": [
                                    "0xPayrollRecipient1...",
                                    "0xPayrollRecipient2...",
                                    "0xPayrollRecipient3...",
                                ]
                            },
                            "review_if": {"amount_usd_gt": "300"},
                        },
                    },
                    {
                        "name": "deny-over-limits",
                        "type": "transfer",
                        "rules": {
                            "effect": "deny",
                            "when": {},
                            "deny_if": {
                                "amount_usd_gt": "500",
                                "usage_limits": {
                                    "rolling_30d": {"amount_usd_gt": "5000"}
                                },
                            },
                        },
                    },
                ],
                "duration_seconds": 7776000,
            },
        )
        pact_id = result["pact_id"]
        print(f"Pact {pact_id} submitted — awaiting owner approval in Human App")

        # Poll for activation
        while True:
            pact = await client.get_pact(pact_id)
            if pact["status"] == "active":
                break
            await asyncio.sleep(10)

        print(f"Pact active. Scoped API key: {pact['api_key']}")

asyncio.run(main())

### Step 3: Owner approves  in Human App

The pact appears in the Human App with the intent, execution plan, recipient allowlist, per-payout cap, approval threshold ($300), and monthly budget. The owner reviews and approves. The agent receives a pact-scoped API key and can begin processing payroll.

### Step 4: Agent operates

With an active pact, the agent processes payroll. Payouts up to $300 execute automatically; payouts between $300 and $500 are paused for owner review; payouts over $500 are denied.

WALLET_UUID=<wallet_uuid>

# Routine payout — within per-tx limit, auto-approved
caw tx transfer "$WALLET_UUID" --to "$DST" --token SETH_USDC --amount 200 --chain SETH

# Mid-tier payout — triggers owner review above $300 threshold
caw tx transfer "$WALLET_UUID" --to "$DST" --token SETH_USDC --amount 400 --chain SETH

# Oversized payout — exceeds $500 per-tx cap, denied
caw tx transfer "$WALLET_UUID" --to "$DST" --token SETH_USDC --amount 600 --chain SETH

# Retry with compliant amount
caw tx transfer "$WALLET_UUID" --to "$DST" --token SETH_USDC --amount 500 --chain SETH
